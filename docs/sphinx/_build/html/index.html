Bienvenido a la documentación de e-cidadania!¶

e-cidadania es una aplicación web destinada a la participación ciudadana. Para ello cuenta con una serie de herramientas clave como son el sistema de propuestas o un perfil avanzado de usuario con geolocalización y mensajería.

Este programa está basado en el framework django y en varias librerías externas como pueden ser LDAP o similares.
